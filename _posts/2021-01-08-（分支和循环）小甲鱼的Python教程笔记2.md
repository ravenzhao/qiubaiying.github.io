施工中的博客

``` Python
# 条件语句中，缩紧决定从属关系
if condition:
    statement(s)
else:
    other statement(s)

# 条件表达式
statement(s) if condition else other statement(s)
```

```Python
if condition1:
    statement(s)1
elif condition2:
    statement(s)2
elif condition3:
    statement(s)3
else:
    other statement(s)
```

```Python
# 循环
while condition:
    statement(s)
else:           #当while的条件不再为真时，执行else语句
    statement(s)
```
