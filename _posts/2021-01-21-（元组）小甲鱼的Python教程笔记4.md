``` Python
# 列表- [元素1，元素2，元素3，……]
# 元组-（元素1，元素2，元素3，……）
>>> rhyme = (1,2,3,4,5,'after')
>>> rhyme = 1,2,3,4,5,'after'
>>> rhyme[0]
1
# 元组是不可变的，因此无法对元组的元素进行修改
# 但是如果该元素是指向一个可变的列表，则可以对列表的元素进行修改
>>> s = [1,2,3]
>>> t = [4,5,6]
>>> w = (s , t)
>>> w[0][0] = 0
>>> w
([0,2,3],[4,5,6])
# 元组可以像列表一样，进行切片操作

# 元组只支持count和index方法
>>>nums = 1,1,2,3,4,5,1
>>>nums.count(1)
3
>>>nums.index(3)
3
# 元组没有推导式，按照列表推导式的形式编写元组，会创建一个生成器
# 省略圆括号有可能会出现问题，因此建议始终保持添加圆括号

# 生成只有一个元素的元组
>>> x = (111,)
# 元组的打包与解包
>>> t = (1,3,'test')
>>> x , y , z = t   #元组的解包（任何序列类型都可以执行此操作）
```
