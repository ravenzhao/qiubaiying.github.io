施工中的博客

``` Python
# 创建函数
>>> def MyFirstFunction():
        # code
# 调用函数
>>> MyFirstFunction()
# 当函数发生调用的时候，会从前方代码中寻找函数
# 函数的参数
>>> def MySecondFunction(num):
        print(num + 'OK!')
# 调用
>>> MySecondFunction('很有精神！')
# 多个参数用，隔开
>>> def add(num1,num2):
        print (num1 + num2)
# 函数的返回值，利用关键字return
>>> def add(num1,num2):
        return (num1 + num2)
```
